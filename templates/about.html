Under the Hood

Upon uploading of the files, your uploads are validated through checking their extensions, file size, and file names to ensure that the system will be able to parse them.

Upon successful validation, the system then determines which loader use through file extension, and then begins to extract the text from within the file(s). After this, the extracted text is compared against a regex in order to mine the citations.

The citations are then stored in a list, of a custom object, processed as Academic or Non-Academic based on a JSON of keywords, and ultimately converted to a JSON and written to an output file. A bar chart showing the breakdown of citations is also created, zipped with the output file, and presented to you for download.
